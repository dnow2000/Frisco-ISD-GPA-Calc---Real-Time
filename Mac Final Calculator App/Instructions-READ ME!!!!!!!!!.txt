1. Download the entire folder and save it to your Mac

2. Place the chromedriver file in the applications folder on your Mac
Hint: Open a finder window and on the left side you will see an applications
folder

3. Double click on the chromedriver file to run it (You only need to do this once)

4. If MacOS gives u a permissions error, open system preferences and click on
the lock icon in the bottom left corner and enter your password

Next, click allow on the chromedriver file

5. Now you should be set to use the GPA Calculator app.


*Note this application DOES NOT collect any user data
