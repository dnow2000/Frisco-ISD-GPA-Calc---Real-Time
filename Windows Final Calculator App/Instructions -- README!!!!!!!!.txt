1. Place the chromedriver file at the correct location on your PC:
(C:\Program Files (x86))

If this step is not completed successfully, the program will not run


2. Click on the GPACALC file to run the calculation


3. Once you've sucessfully tested the app, you can remove it from this folder
and place it wherever you want. Just be sure not to delete the chromedriver file 
from where you saved it.


Note: This app does not store userdata or information as can be seen by the source code
on my Github @ github.com/SumitNalavade
